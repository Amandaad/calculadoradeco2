# 🌱 Calculadora de CO₂

Este projeto é uma calculadora de emissões de dióxido de carbono (CO₂), criada para estimar o impacto ambiental de diferentes atividades e ajudar na conscientização sobre sustentabilidade.

## 🚀 Funcionalidades
- Cálculo de emissões de CO₂ baseado em dados de transporte, energia e consumo.
- Interface simples e intuitiva.
- Resultados apresentados em tempo real.
- Possibilidade de comparar diferentes cenários.

## 🛠️ Tecnologias utilizadas
- **Front-end:** HTML, CSS, JavaScript
- **Back-end (opcional):** Node.js / Express
- **Banco de dados (opcional):** SQLite ou MongoDB
